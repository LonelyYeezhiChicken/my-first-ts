# 字串 - string

## 定義:
- 可定義任意文字
- 怎麼輸入就怎麼顯示

### ts 宣告
```ts
let myName:string ='Chicken';
let sayHi:string = ' say hi!';
```

### js 宣告
```js
var myName = 'Chicken';
var sayHi = ' say hi!';
```
