# 數值 - number

## 定義:
- 各式數字

### ts 宣告
```ts
let one:number = 1;
let two:number = 2;
let infinityNumber: number = Infinity;
```

### js 宣告
```js
var one = 1;
var two = 2;
var infinityNumber = Infinity;
```

### wy 宣告
```wy
吾有一數。曰一。名之曰「第一」
吾有一數。曰二。名之曰「第二」
吾有一物。曰「Infinity」。名之曰「infinityNumber」
```
